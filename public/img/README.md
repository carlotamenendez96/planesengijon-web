# Imágenes estáticas — public/img/

Todos los archivos de esta carpeta se sirven directamente bajo `/img/` en la URL.

## Imágenes necesarias

| Archivo | Usado en | Tamaño recomendado |
|---|---|---|
| `hero-san-lorenzo.webp` | Hero de la home | 1920×1080 |
| `verano.webp` | Card temporada Verano | 800×600 |
| `otono.webp` | Card temporada Otoño | 800×600 |
| `invierno.webp` | Card temporada Invierno | 800×600 |
| `primavera.webp` | Card temporada Primavera | 800×600 |
| `placeholder-itinerary.webp` | Fallback ItineraryCard | 600×400 |
| `placeholder-beach.webp` | Fallback BeachCard | 600×400 |
| `placeholder-restaurant.webp` | Fallback RestaurantCard | 400×260 |
| `placeholder-place.webp` | Fallback PlaceCard | 400×280 |

## Cómo añadir imágenes

Coloca los archivos directamente en esta carpeta (`public/img/`).
No hace falta importarlas en el código — Nuxt las sirve automáticamente.

Recomendaciones:
- Formato **WebP** siempre que sea posible (mejor compresión)
- Nombres en minúsculas con guiones (no espacios ni mayúsculas)
- Comprimir antes de subir con https://squoosh.app
